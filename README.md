##

### Installation

- Just, Clone this repository -
  ```
  git clone https://github.com/Abid0987/joke.git
  ```

- Now go to cloned directory and run `joke.py` -
  ```
  $ cd joke
  $ python3 joke.py
  ```

- On first launch, It'll install the dependencies and that's it. ***joke*** is installed.
